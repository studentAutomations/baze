SQL Code Example

SELECT *
FROM RADNIK
WHERE MATBRS IS NULL

Code Explanation

The SQL statement retrieves all records from the RADNIK table where the column MATBRS is NULL.

The Issue with NULL

In SQL, you cannot use standard comparison operators to check for NULL. Instead, use IS NULL.

Corrected Code

To properly filter for rows where MATBRS is NULL, the correct syntax is:

SELECT *
FROM RADNIK
WHERE MATBRS IS NULL
